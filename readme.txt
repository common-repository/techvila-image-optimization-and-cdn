Completely automatic image optimization and load static rerouces from cdn

== Description ==

This plugin will optimize your images to webp version using techvila.com server. There will be no load on your server. It will also load your static resources like images, js, css from cloudflare cdn. It will dramatically improve your website speed.

There is no sign up and no configuration needed. Just activate the plugin. When user will visit the front end of your site, the images, js and css will be loaded from techvila.com server, images will be converted to webp and served in browser that supports webp.

Learn more about techvila cdn https://techvila.com/free-cdn
Here is their terms of service https://techvila.com/privacy-policy/

Plug and play. No configuration needed.


== Installation ==

1. Download the plugin

2. upload to plugin directory

3. activate from dashboard

4. There is no settings

5. Your site images and static resource like js and css will be loaded from cdn.
